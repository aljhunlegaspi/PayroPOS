## Project Overview

### What is PayroPOS?

PayroPOS is a modern Point of Sale (POS) system designed for small to medium retail stores. It combines traditional POS functionality with a built-in customer credit tracking system (commonly known as "utang" system in the Philippines).

### Key Features

- **Multi-platform**: Web admin dashboard + Android mobile app
- **Barcode/QR Scanning**: Quick product lookup and checkout
- **Customer Credit System**: Track customer debts and payments
- **Multi-user Support**: Store owner, staff, and customer roles
- **Real-time Sync**: Powered by Firebase Firestore
- **Digital Receipts**: Generate and share receipts instantly

### Target Users

| User Type | Description |
|-----------|-------------|
| Super Admin | Platform administrator (manages all stores) |
| Store Owner | Business owner (manages their store) |
| Store Staff | Cashier/employee (processes sales) |
| Customer | Buyer (optional app access for credit tracking) |

### Firestore Security Rules

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    // Users collection
    match /users/{userId} {
      allow read: if request.auth != null;
      allow write: if request.auth.uid == userId;
    }

    // Stores collection
    match /stores/{storeId} {
      allow read: if isStoreOwnerOrStaff(storeId);
      allow write: if isStoreOwner(storeId);

      // Subcollections
      match /categories/{categoryId} {
        allow read: if isStoreOwnerOrStaff(storeId);
        allow write: if isStoreOwner(storeId);
      }

      match /products/{productId} {
        allow read: if isStoreOwnerOrStaff(storeId);
        allow write: if isStoreOwner(storeId);
      }

      match /customers/{customerId} {
        allow read: if isStoreOwnerOrStaff(storeId);
        allow write: if isStoreOwnerOrStaff(storeId);
      }

      match /transactions/{transactionId} {
        allow read: if isStoreOwnerOrStaff(storeId);
        allow create: if isStoreOwnerOrStaff(storeId);
        allow update, delete: if isStoreOwner(storeId);
      }

      match /creditPayments/{paymentId} {
        allow read, write: if isStoreOwnerOrStaff(storeId);
      }

      match /staff/{staffId} {
        allow read: if isStoreOwnerOrStaff(storeId);
        allow write: if isStoreOwner(storeId);
      }
    }

    // Helper functions
    function isStoreOwner(storeId) {
      return get(/databases/$(database)/documents/stores/$(storeId)).data.ownerId == request.auth.uid;
    }

    function isStoreOwnerOrStaff(storeId) {
      return isStoreOwner(storeId) ||
             exists(/databases/$(database)/documents/stores/$(storeId)/staff/$(request.auth.uid));
    }
  }
}
```
